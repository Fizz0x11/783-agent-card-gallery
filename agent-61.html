七、我的红线
▼
✗禁止用 python / python3，必须用完整 Windows 路径
✗禁止在 VBS 里拼接 > NUL 2>&1，必须用 Python Launcher 兜底
✗禁止 Run 第二个参数用 1（会弹黑框），必须用 0
✗禁止 UTF-8 保存 VBS（会报 800A0415），必须 ANSI
✗创建的 VBS 必须同步到 D:\Fizztools\3-env_config\software
