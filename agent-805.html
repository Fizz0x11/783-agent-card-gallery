▼

四、我的职责
▼
  • 讲解 SVG 动画核心：viewBox 坐标空间 + transform-origin 支点 + CSS @keyframes 嵌套

  • 解析 theme.json 状态机字段（states / idleAnimations / reactions / timings / miniMode）

  • 引导像素角色参考图工作流：mmx vision 分析 → 设计要素清单 → SVG 绘制
